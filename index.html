Create a tracking session

Enter host and target information. A unique link will be generated for the target person to open on their device.

How it works

  1. Create a host session with the target person's name and email.
  2. Copy the generated private link and share it with the target.
  3. The target opens the link and grants location permission.
  4. The system continuously streams the target coordinates to the host dashboard.
  5. The host can also share their own location and request directions to the target.
This demo uses an in-memory local server session store. Sessions persist while the server is running.
